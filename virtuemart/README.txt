#Инфо
Плагин: InterKassa Gateway
Описание: Платежный шлюз "Интеркасса" для сайтов Joomla 3.4.8 + VirtueMart 3.0.x (версия протокола Интеркассы 2.0)
Предназначение: Модуль добавляет способ оплаты "Интеркасса" для Open VirtueMart 3.0.x

#Установка
1. Скачать плагин к себе на компьютер, распаковать
2. Закачать все из папки на сервер
3. зайти в админку, добавить метод оплаты, выбрать процесс pm_interkassa, настроить модуль в соответствии со своим аккаунтом

ВАЖНО!
На сайте Интеркассы в ЛК нужно в настройках "Безопасность" включить под урлами успешной оплаты и файл переопределние запроса!
Жмем "Сохранить"

Метод активен и появится в списке оплат вашего магазина.
